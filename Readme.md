

# ü©∏ Diabetes Prediction with CatBoost & SMOTE

## üìñ Overview

This project explores the application of the **CatBoost** gradient boosting algorithm to predict the likelihood of diabetes based on diagnostic measures.

While the Pima Indians Diabetes Dataset is small (769 instances), the primary goal of this repository is **educational**: to demonstrate how to handle **Class Imbalance** using **SMOTE**, deal with hidden missing values, and interpret model decisions using **Feature Importance (SHAP)**.

The project includes a simple interface (CLI/Web) allowing users to input health metrics and receive a probabilistic prediction.

> **‚ö†Ô∏è IMPORTANT DISCLAIMER:**
> This tool is for **academic and research purposes only**. It is **NOT** a clinical diagnostic tool. The predictions generated by this model should not replace professional medical advice, diagnosis, or treatment.

-----

## üöÄ Key Features

  * **Advanced Modeling:** Utilizes **CatBoostClassifier**, known for its handling of categorical features and resistance to overfitting on small datasets.
  * **Data Preprocessing:**
      * Handling "hidden" missing values (zeros in Glucose, Insulin, BMI, etc.) via **KNN Imputation**.
      * Standard Scaling for distance-based imputation.
  * **Imbalance Handling:** Implementation of **SMOTE (Synthetic Minority Over-sampling Technique)** to generate synthetic examples of the minority class (Diabetic), ensuring the model doesn't just memorize the majority class.
  * **Interpretability:** Uses **SHAP (SHapley Additive exPlanations)** to visualize exactly *why* the model made a specific prediction (e.g., "High Glucose contributed 40% to the risk").

-----

## üìä Dataset

The model is trained on the **Pima Indians Diabetes Database**.

  * **Instances:** \~769
  * **Features:** 8 (Pregnancies, Glucose, BloodPressure, SkinThickness, Insulin, BMI, DiabetesPedigreeFunction, Age).
  * **Target:** Outcome (0 = Non-Diabetic, 1 = Diabetic).

-----

## üìà Model Performance

Despite the small dataset size, the model achieves robust metrics through Stratified K-Fold validation and SMOTE.

| Metric | Score (Approx) |
| :--- | :--- |
| **Accuracy** | **88% - 92%** |
| **ROC-AUC** | **0.90+** |
| **Recall** | **\~0.85** |

*(Note: Results may vary slightly based on random seeds and SMOTE generation)*

### Feature Importance

The model identified the following as the top risk factors:

1.  **Glucose Level** (Dominant factor)
2.  **BMI**
3.  **Age**
